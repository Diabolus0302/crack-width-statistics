# Changelog

All notable changes to this project are documented in this file.

## [0.2.0] - 2026-07-16

### Added

- Length-weighted mean, median, P10, P90, maximum, and standard deviation of
  crack width for individual images and the complete dataset.
- Over-threshold crack-length percentage.
- Separate crack-segment, width-interval, and quality-control tables.
- Width-bin percentages, cumulative percentages, and above-lower-bound
  percentages.
- Branch-point detection and branch-split review segments while preserving
  merged-network length.
- Quality-control metrics for crack area, skeleton points, filtered short
  objects, endpoints, branch points, isolated points, and out-of-range widths.
- Width-bin legend and table-driven highlighting of segments and width slices.
- Mouse-wheel zoom, drag-to-pan, double-click fit, overlay opacity, and line
  width controls.
- Batch quality-preview window and exported batch quality image.
- Detailed English user guide and updated software screenshot.

### Changed

- Expanded the Excel export with per-image summaries, pooled width-bin
  statistics, image overview, and global width statistics.

## [0.1.0] - 2026-05-28

- Initial public release.
